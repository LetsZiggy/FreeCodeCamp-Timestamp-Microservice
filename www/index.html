API Basejump: Timestamp Microservice

User stories:

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date.
    (example: 15 Jan(uary) 2016 || Jan(uary) 15 2016 || 2016 Jan(uary) 15)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date
  3. If it does not contain a date or Unix timestamp, it returns null for those properties

Example usage:

https://letsziggy-freecodecamp-timestamp-microservice.glitch.me/15%20January%202016
https://letsziggy-freecodecamp-timestamp-microservice.glitch.me/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15 2015" }
